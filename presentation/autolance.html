BRIGHT DATA INTEGRATION
04 / 09

▸ Powered by Bright Data · Web Unlocker

Bypassing Cloudflare to
scrape live Upwork data

Upwork is heavily Cloudflare-protected and JavaScript-rendered. Standard scrapers are blocked instantly. Bright Data's Web Unlocker routes requests through residential IPs, returns fully rendered HTML — unlocking data that was previously impossible to access programmatically.

🔓
Web Unlocker API [zone1]
Bypasses Cloudflare bot detection via residential IP rotation. Returns 1.5MB+ fully rendered Upwork search pages with embedded job state.
🧩
__NUXT__ State Parser (Custom)
Decodes Upwork's IIFE-compressed __NUXT__ variable table to extract structured job data: title, skills, budget, dates, client info.
⚡
Parallel Fetch + Redis HTML Cache
3 concurrent Unlocker requests. 14-min cache TTL eliminates redundant API calls. Profile-driven keywords from user's CV guide which URLs are fetched.
◈ SCRAPE PIPELINE
CV Profile parsed → niche + top 3 skills extracted
BD Bright Data Unlocker API → Upwork search page
HTML 1.5MB rendered HTML received (bypasses Cloudflare)
NUXT __NUXT__ IIFE decoded → 10-40 job objects extracted
DB Jobs ingested → client analyzed → bid strategy computed
AI Claude scores each job → alerts dispatched
